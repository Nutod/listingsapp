## Run Locally

Clone the project

```bash
  git clone https://github.com/Nutod/listingsapp
```

Go to the project directory

```bash
  cd listingsapp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Running Tests

To run tests, run the following command

```bash
  npm run test
```
